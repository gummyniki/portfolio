title: The Future

Hello everyone! Today's blog post won't be *too* technical but instead will be discussing the *future* of this engine and what will come next.

---

So as I've previously said, I started learning graphics almost *2 years ago* and started making this engine almost *a year ago*, and it has been mostly the only thing I've worked on.

It has been really fun and I've been enjoying every moment, but at some point everything gets boring.
Maybe it's because I've worked on the same thing for a year, or maybe it's because I've done the same thing for about 2 years, but it's kinda lost it's charm.

As I've stated last devlog, this engine is coming to an end, and after the final features I've planned are done, I will be taking a break from this project and probably graphics as a whole.

This is not to say I won't ever be coming back to this project, on the contrary, I'm almost **certain** I will come back to this project sometime, but for my own sake and this project's, it's first season will be reaching it's end soon.

Now you might be wondering what I will be posting on here, for the next few weeks, new blogs about this engine will continue, but after that, i'm not sure.

I have a lot of ideas, I will most likely be starting **another** project and documenting that, but maybe I'll be tackling a whole new subject, *who knows.*

---

Let me also talk a bit about the engine's progress. One of the *2* major bugs I've talked about last devlog have been fixed, and I'm starting work on the second one.

After that, I will either continue working on the physics engine for a bit more, or add some kind of **scene loader**, but I'm not sure if I have the energy to do that right now.
I won't go into too much detail about the bug I solved as that's for the next devlog. I just wanted to talk about what will come next since my main project is slowly coming to an end.

I think that will be it for today, I will see you sometime next week, **see ya!**
